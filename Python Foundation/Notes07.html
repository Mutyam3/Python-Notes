            ==> 1. Introduction to Tuples: 
                   ----------------------- 


                   An Ordered, immutable collection of items 
                   items can be of different data types 
                   Once created, elements can't be changed 
                   enclosed with parentheses
                   support indexing

                 Ex : (1, 'hello', 3.14)
                      ('orange', 'grape')

                
            ==> 2.  Creating a tuple using tuple literals : 
                    ---------------------------------------- 

                    p = (3,4,5)

                    b = 4,7,5 
                    print(b) // (4,7,5)  
                    Note : In lists we should always enclosed with square brackets 
                           In tuples even we don't give the parentheses python considered it as tuple but we have to give the 
                           parentheses. but tuple need atleast one comma

                    
            ==> 3.  Creating a tuple by explicit typecasting: 
                    ----------------------------------------- 
                    
                    l = [1,2,3]
                    p = tuple(l)
                    print(p) // (1,2,3)

            ==> 4.  Creating a singleton tuple : 
                    -------------------------- 

                    p = 4,   # singleton tuple 
                    print(p)
                    print(type(p))


                    output : 
                    (4,)
                    < tuple >

                    Note :  Tuple needs atleast one comma to consider as a tuple 

                    Ex : p = (4) 
                         print(p) // 4 it consider as a int not tuple 
                         print(type(p)) // < int >

            ==> 7.  Positive and negative index : 
                    ----------------------------- 

                    print(t[0]) // 14 
                    print(t[1]) // 15 
                    print(t[2]) // 80 
                    print(t[3]) // 22 
                    print(t[4]) // 65 

                    print(t[-1]) // 65
                    print(t[-2]) // 22 
                    print(t[-3]) // 80 
                    print(t[-4]) // 15 
                    print(t[-5]) // 14 

                    print(t[len(t)-1]) // t[4] ==> 65 
 
            ==> 8. IndexError : 
                   ------------- 

                    t = ( 1, 2, 3 4)

                    print(t[4]) // tuple index out of range 

            ==> 9. Membership testing : 
                   -------------------- 

                   t = (5, 2 , 8 , 6 , 4 , 1)

                   print(3 in t) // False 

                   print(4 in t) // True 

                   print(9 not in t) // True 

                   print(5 not in t) // False 

            ==> 10. The len() function : 
                    -------------------  

                    t = (2, 5 , 8 , 7 , 6 , 9)

                    print(len(t)) // 6 

            ==> 11. Immutability :
                    -------------- 

                    tuple is like a sealed container
                    
                    t = ( 4, 6, 3)
                    t[1] = 10 

                    print(t) // 'tuple' object doesnot support item assignment 

            ==> 12. Tuple slicing : 
                    --------------

                    t = (5, 1, 6, 8, 9, 7)

                    print(p[0 : 6])  // (5,1,6,8,9,7)

                    print(p[:6]) // (5,1,6,8,9,7) 

                    print(p[:]) //(5,1,6,8,9,7) 

                    print(p[2:5]) // (6,8,9) 

            ==> 13. Concatenation in tuples : 
                    -------------------------- 

                    p = (1, 3, 5)
                    q = (8 , 41, 50)

                    print(p + q) // (1, 3 , 5 , 8 , 41 , 50)

                Note : '+' operator concate every sequence datatype like string, list and tuples 

            ==> 14. Nested Tuples : 
                    ---------------- 

                    p = (4, 12, 8 , 9 , (7, 6, 5))

                    print( p[0]) // 4 ==> int datatype
                    print( p[1]) // 12 ==> int 
                    print( p[2]) // 8 ==> int
                    print( p[3]) // 9 ==> int 

                    print(p[4]) // (7,6,5) ==> tuple 

                    print(len(p)) // 5

                    print(len(p[4])) // 3 

                    print(p[4][0]) //7 
                    print(p[4][1]) //8 
                    print(p[4][2]) //9 

            ==> 15. Looping in tuples : 
                    ------------------- 

                    t = (10, 50, 85, 70, 65)

                    while loop :

                    i = 0 

                    while i < len(t):
                         print(t[i])
                         i = i + 1
                         
                    output : 

                        10
                        50 
                        85
                        70
                        65

                    for loop : 

                      for i in t :
                          print(i)

                          output : 

                          10
                          50 
                          85
                          70
                          65
