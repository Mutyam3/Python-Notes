            ==> 12. Class and instance variables : 
                    ------------------------------ 

                    class Person():
                        country = 'India' // class Variable /class attribute which is common for all 

                        def __int__(self, a,b,c):
                             self.firstname = a   // here firstname, age and city are the instance variables as when we create instances and gives the values then only it gets filled
                             self.age = b 
                             self.city = c

                    person1 = Person("Mutyam", 50, 'hyd')  #class instantiation
                    person2 = Person('Bhargav', 25, 'Vizag')
                    person3 = Person('Balu', 42, 'Chennai')

                    Class variables serve as shared attributes across all instances of a class, 
                    providing a centralized means for managing data common to the entire class.


                    On the other hand, instance variables encapsulate unique characteristics for each object,
                    maintaining individuality.


                 class Dog():
                     type = 'animal'

                    def __init__(self, breed, color):
                             self.breed =  breed 
                             self.color = color 

                dog1 = Dog('Labrador', 'Black')
                dog2 = Dog('Golden retriever', 'Brown')

                print(dog1.type) // animal
                print(dog1.breed) // Labrador
                print(dog1.color) // Black

                print(dog2.type) // animal
                print(dog2.breed) // Golden Retriever
                print(dog2.color) // Brown 


                unique characteristics write under init function which are instance variables. 
                common things write under class which are class variables. 


                init is the function name
