              ==> Introduction to Functions :
                  -------------------------- 

                  ** A function is a block of code which only runs when it is called.

                  ** You can pass data, known as parameters, into a function.

                  ** A function can return data as a result.


              ==> 1. Importance of Functions : 
                     ------------------------- 

                     Functions are useful for two reasons. 

                     First, functions allow you to write a block of code once and use it multiple times. 
                     You only need to write and debug the code once. This saves you lots of time in the future. 
                     Functions also make your code easier to maintain because, if you need to update your function at some point,
                     you only need to edit it in one place.  [Reusability]
                     
                     Secondly, writing functions can make your code easier for the reader to understand.  [ Readability ]
                     If your function has a descriptive name (e.g. a function designed to calculate the tip on a restaurant bill could be called calculate_tip) 
                     then anybody reading your code can understand what each line does without having to make sense of the calculations.

              ==> 6. Default argument values : 
                     ------------------------ 

                     def printsum(a,b,c)
                         return a + b + c 

                     x = printsum(10,2,8,7) this is always wrong passing more arguments than parameters 
                     print(x) // error ==> printsum() takes 3 positionsal argument but 4 were given
                     
                     ---------------------------------------------

                     def printsum(a,b,c)
                         return a + b + c 

                     x= printsum(10,2) 
                     print(x) // error ==> printsum() missing 1 required positionalargument : 'c'

                     ------------------------------------------------------

                     def printsum(a,b,c=0)
                         return a + b + c 

                    x = printsum(10,2)
                    print(x) // 12 

                      def printsum(a,b,c=0) // this way of telling to python that if the value is not there please take this value as default value known as default argument value 
                           return a + b + c 

                    x = printsum(10,2,3)
                    print(x) // 15

                    Note : 

                    def printsum(a=0,b,c=0) here donot write a = 0 because non-default argument follows default arguments 
                       return a + b + c 

                    x = printsum(10,2)
                    print(x) // 12 

                    (a, b=0, c=0) this one is correct 
                    (a, b=0, c) this one is wrong as after default not follows non default 
                    (a=1,b=2, c=4) this one is also correct making all default arguments

              ==> 11. Variable scope : 
                      ---------------- 

                      In Python, a scope defined the region of a program where a particular variable lives and can be accessed
                      Think of scopes as containers that hold variables. 
                      There are two primary types of scope : 

                            ==> Local Scope  
                            ==> Global Scope
                            
                    Local Scope : 
                        A local scope is created whenever a function is called. 
                        It exists only within that specific function's body. 
                        Variables declared inside a function are local variables 
                        They are born when the function starts and vanish when the function ends. 
                        You can only access a local variable from within the function where it's defined. 
                        Trying to use it outside the function results in a Name Error. 

                        ex : 

                        def abc(): 
                            z = 50 

                        print(z) // z is not defined u can't access the local variable z from outside that function 

                        x = 10 
                        y = 20 

                        def abc():
                            print(x)
                            print(y)
                            z = 10 

                         abc() // 10
                                  20
