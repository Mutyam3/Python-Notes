             11. match case : 
                 -------------- 

                    Just for the syntax instead of writing multiple elif statements 
                    In python less use of this match case 
                    In this match case we don't have break statement 
                            
                            color = input('Enter your color : ')

                                match color: 
                                    case 'red' : 
                                        print('red')
                                    case 'blue' : 
                                        print('blue')
                                    case 'yellow' : 
                                        print('yellow')
                                    case 'green' : 
                                        print('green')
                                    case _:             ====> default case 
                                         print('invalid color')
