                ==> 16. The enumerate() function :

                ==> 17. List methods : 
                    ------------------- 

                         nums = [23, 12, 67, 44, 89, 90, 36]

                         list methods : 

                          1. append() :    add a value at the end 
                            ----------   takes only one argument at the end 

                                nums.append(99)
                                print(nums) // [23, 12, 67, 44, 89, 90, 36, 99] 

                                nums.append(99, 50)
                                print(nums) // error 

                          2. insert() :     add a value at specific index 
                             ---------    first argument is index and second argument is a value
                                          it won't remove any value 

                                  nums.insert(3, 10) 
                                  print(nums) // [23, 12, 67, 10, 44, 89, 90, 36]
                                  nums.insert(0,32) 
                                  print(nums) // [32,23, 12,67,10,44,89,90,36]

                          3. remove() :    it remove an element / item from a list 
                             ---------   finding and remove anywhere 

                                   nums.remove(90)
                                   print(nums)  // [23,12,67,44,89,36]

                          4. pop() :      remove last element or value 
                            ------      given index as an argument it gets removed that index value  


                                    nums.pop()
                                    print(nums) // [23,12,67,44,89,90]

                                    nums.pop(1)
                                    print(nums) //[23, 67, 44, 89, 90]

                          5. Count() : 
                             -------      helps to count the values how many times it coming  

                                     nums = [12, 55, 90, 57, 44, 25, 66, 44]

                                     x = nums.count(44)
                                     print(x) // 2 

                                     y = nums.count(90)
                                     print(y) // 1 

                           6. clear() : 
                             ---------   it clears the whole list and make it as empty list 

                                         nums.clear()
                                         print(nums) // []

                           7. sort(): 
                              --------    sort method makes the items in list to be in ascending order 

                                        nums = [20, 4, 5, 12, 10, 80, 15, 35]

                                        nums.sort()
                                        print(nums) // [4,5,10,12,15,35,80]

                            8. reverse() : 
                               -----------   arrange the items from last index to first index as reverse 

                                        nums = [20, 4, 5, 12, 10, 80, 15, 35] 

                                        nums.reverse()
                                        print(nums) // [35, 15, 80, 10, 12, 5, 4, 20]

                                to arrange the items in descending order we can use sort() and reverse() methods 
                                as sort() method arrange the items in ascending order and reverse() method arrange the items in reverse direction 

                                        nums.sort()
                                        nums.reverse()

                                        print(nums) // [80, 35, 15, 12, 10, 5, 4]
