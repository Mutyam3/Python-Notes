                    -------------------

                16. Looping in Strings :
                    -------------------- 

                    we can use both while and for loops for looping in strings 

                    while loop : 
                    ------------ 

                    a = "Hello"

                    i = 0 

                    while i<= len(a)-1:   or i < len(a)
                          print(a[i])
                          i = i + 1 

                    output : 
                    -------- 

                    H 
                    e 
                    l 
                    l 
                    o 

                    for loop : 
                    ----------- 

                    a = "Hello"

                    for i in a: 
                        print(i)

                    here i has direct characters not index 
                    when we working with sequence data types like lists | tuples | strings , we use for loop because its easy

                  ==> Search all occurrences of a character in a string 

                    s = "Apple" 

                    i = 0

                    while i < len(s):
                         if s[i] == 'p' :
                            print(i)
                         i = i + 1 

                    Output : 
                    ---------
                    1 
                    2 

                  ==> Count Occurences of a character in a string 

                     s = "Apple" 

                     i = 0 
                     count = 0

                     while i < len(s) : 
                           if s[i] == 'p':
                              count = count + 1
                           i = i + 1 

                    output : 
                    -------- 
                    2 

                  ==> Delete all Occurrence of a character in a string 

                       s = "MutyaM"

                       new = ""

                       i = 0
                       
                       while i < len(s): 
                           if s[i] != 'M'
                              new = new + s[i] 
                         i = i + 1 

                      print(new) // utya
                      
                  
                  ==> Remove First Occurrence of a character in a string 

                        s = "MutyaM"

                        new = ""

                        i = 0 
                        count = 0

                        while i < len(s):
                             if s[i] == 'M':
                                count = count + 1 
                             if (s[i] == 'M' and count > 1) or (s[i] != 'M') :
                                 new = new + s[i]

                        i = i + 1 

                        print(new) // utyaM

                        
                  ==> Reverse a String : 

                  s = "racecar"
                  new = ""
                  i = len(s)-1
                  while i >= 0:
                     new = new + s[i]
                     i = i - 1
                  
                  print(new)
                  if new == s:
                     print('Its a palindrome string')
                  else:
                     print("It's not a palindrome string")
